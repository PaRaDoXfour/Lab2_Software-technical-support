# Library Management System - Бакалаврська робота

[![Ukrainian](https://img.shields.io/badge/Language-Ukrainian-blue.svg)](#)
[![English](https://img.shields.io/badge/Language-English-red.svg)](#)

Цей репозиторій містить лендінг бакалаврської роботи на тему: **"Інформаційна система для управління бібліотекою з підтримкою різних типів медіаресурсів"**.

## 📌 Про проєкт

Лендінг сторінка розроблена для презентації кваліфікаційного проєкту факультету Електроніки та інформаційних технологій (ЕлІТ) Сумського державного університету (СумДУ). Вона описує ключові аспекти розроблюваної системи:

*   **Актуальність теми:** Потреба в оптимізації обліку та каталогізації не лише класичних друкованих видань, але й цифрових медіаресурсів.
*   **Мета та завдання:** Розробка комплексної інформаційної системи для автоматизації процесів управління бібліотечним фондом.
*   **Очікувані результати:** Зменшення часу обслуговування, єдиний центр управління, підвищення безпеки даних.
*   **Методологія та стек технологій:** React.js, Node.js + Express.js, PostgreSQL, JWT аутентифікація.

## 🚀 Технології сторінки (Landing Page)

Сама сторінка (frontend) побудована з використанням сучасних веб-технологій та підходів:

*   **HTML5:** Семантична розмітка для кращої доступності та SEO.
*   **Tailwind CSS (через CDN):** Швидка та сучасна стилізація, адаптивний дизайн (Responsive Web Design).
*   **Vanilla JavaScript:** Реалізація перемикача мов (Українська / English) та інтерактивності без важких фреймворків.
*   **SEO & Accessibility (a11y):** Вбудований Schema.org JSON-LD, Open Graph теги, ARIA-атрибути для скрінрідерів.
*   **Шрифти:** Google Fonts (Inter).

## 🌍 Багатомовність (i18n)

Лендінг підтримує дві мови:
1.  **Українська** (за замовчуванням)
2.  **English**

Перемикання відбувається миттєво (на стороні клієнта) за допомогою JavaScript, який ховає/показує відповідні `<span>` елементи з атрибутом `data-lang`.

## 🛠 Запуск проєкту локально

Сторінка є повністю статичною (без серверного рендерингу). Ви можете переглянути її кількома способами:

### Спосіб 1: Пряме відкриття (Найпростіший)
Просто скачайте репозиторій і двічі клікніть на файл `index.html`, щоб відкрити його у вашому веб-браузері.

### Спосіб 2: Через локальний сервер (Рекомендований)
Для кращого досвіду та уникнення можливих проблем з CORS, рекомендується використовувати локальний веб-сервер.

**Використання Python (вбудований у macOS/Linux):**
```bash
# Перейдіть у папку з проєктом
cd шлях/до/папки/Lab2_Software-technical-support

# Запустіть сервер
python3 -m http.server 8000
```
Відкрийте у браузері: `http://localhost:8000`

**Використання Node.js (npx serve):**
```bash
# Перейдіть у папку з проєктом і запустіть serve
npx serve .
```
