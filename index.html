Актуальність темиRelevance of the Topic

В епоху стрімкої цифровізації зберігання та обробка інформації у традиційних бібліотеках стає рутинною та неефективною. Впровадження сучасної Інформаційної системи управління бібліотекою (Library Management System) дозволяє суттєво оптимізувати процеси обліку, автоматизувати видачу та спростити каталогізацію не лише класичних друкованих видань, але й цифрових медіаресурсів (електронних книг, аудіо- та відеоматеріалів). Це забезпечує швидкий та зручний доступ студентів і викладачів до необхідної бази знань.

In the era of rapid digitalization, storing and processing information in traditional libraries becomes routine and inefficient. The implementation of a modern Library Management System allows to significantly optimize accounting processes, automate loaning, and simplify cataloging not only for classic printed publications, but also for digital media resources (e-books, audio, and video materials). This ensures fast and convenient access for students and teachers to the required knowledge base.

Мета та завданняGoals and Tasks

Метою бакалаврської роботи є розробка комплексної інформаційної системи для автоматизації процесів управління бібліотечним фондом та покращення клієнтського досвіду читачів.

The goal of the bachelor's thesis is to develop a comprehensive information system to automate the processes of managing the library fund and improving the reader's customer experience.

1

АналізAnalysis

Дослідження предметної області, існуючих аналогів та вимог до сучасних бібліотечних інформаційних систем. Research of the domain, existing analogs, and requirements for modern library information systems.

2

Проектування БДDB Design

Розробка надійної реляційної схеми бази даних для збереження інформації про користувачів, книги та медіаресурси. Development of a reliable relational database schema to store information about users, books, and media resources.

3

Пошукова підсистемаSearch Subsystem

Реалізація ефективних алгоритмів пошуку за авторами, категоріями, тегами та ключовими словами. Implementation of efficient search algorithms by authors, categories, tags, and keywords.

4

Розробка модулівModule Development

Створення модуля адміністратора (управління фондом) та модуля читача (перегляд, електронне бронювання). Creation of the administrator module (fund management) and the reader module (viewing, e-booking).

5

ТестуванняTesting

Проведення комплексного модульного та інтеграційного тестування функціоналу для забезпечення безпеки і стабільності роботи системи. Conducting comprehensive unit and integration testing of functionality to ensure the security and stability of the system.

Очікувані результатиExpected Results

Зменшення часу обслуговуванняReduced Service Time

Автоматизація процесів дозволить скоротити час видачі та приймання літератури до 40%.Process automation will reduce literature issuing and receiving time by up to 40%.

Єдиний центр управлінняUnified Management Center

Глобалізований доступ до друкованих і цифрових медіаресурсів з одного інтерфейсу.Globalized access to printed and digital media resources from a single interface.

Підвищення безпеки данихEnhanced Data Security

Впровадження JWT-аутентифікації та розмежування ролей збереже дані від несанкціонованого доступу.Implementation of JWT authentication and role segregation will protect data from unauthorized access.

Методологія та стек технологійMethodology and Tech Stack

Frontend

React.js для створення динамічного та адаптивного інтерфейсу користувача (SPA) з використанням компонентного підходу.React.js for creating a dynamic and responsive user interface (SPA) using a component approach.

Backend

Node.js із фреймворком Express.js для побудови швидкого та масштабованого RESTful API.Node.js with the Express.js framework for building a fast and scalable RESTful API.

Database

PostgreSQL для структурованого та захищеного зберігання об'ємної інформації бібліотеки.PostgreSQL for structured and secure storage of voluminous library information.

Tools

JWT для безпечної аутентифікації користувачів, та Git для контролю версій і командної роботи.JWT for secure user authentication, and Git for version control and teamwork.
