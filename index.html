Інформаційна система управління бібліотекою з підтримкою медіаресурсів

Information system for library management with support for media resources

Актуальність теми

В епоху стрімкої цифровізації зберігання та обробка інформації у традиційних бібліотеках стає рутинною та неефективною. Впровадження сучасної Інформаційної системи управління бібліотекою (Library Management System) дозволяє суттєво оптимізувати процеси обліку, автоматизувати видачу та спростити каталогізацію не лише класичних друкованих видань, але й цифрових медіаресурсів (електронних книг, аудіо- та відеоматеріалів). Це забезпечує швидкий та зручний доступ студентів і викладачів до необхідної бази знань.

Мета та завдання

Метою бакалаврської роботи є розробка комплексної інформаційної системи для автоматизації процесів управління бібліотечним фондом та покращення клієнтського досвіду читачів.

Основні завдання:

  • 1. Аналіз: Дослідження предметної області, існуючих аналогів та вимог до сучасних бібліотечних інформаційних систем.
  • 2. Проектування БД: Розробка решти реляційної схеми бази даних для надійного збереження інформації про користувачів, книги та медіаресурси.
  • 3. Пошукова підсистема: Реалізація ефективних алгоритмів пошуку за авторами, категоріями, тегами та ключовими словами.
  • 4. Розробка модулів: Створення та впровадження модуля адміністратора (управління фондом) та модуля читача (перегляд, електронне бронювання).
  • 5. Тестування: Проведення комплексного модульного та інтеграційного тестування функціоналу для забезпечення безпеки і стабільності роботи системи.

Методологія та стек технологій

Проєкт реалізується на основі сучасних веб-технологій з використанням архітектури клієнт-сервер:

  • Frontend (Клієнтська частина): React.js для створення динамічного та адаптивного інтерфейсу користувача (SPA) з використанням компонентного підходу.
  • Backend (Серверна частина): Node.js із фреймворком Express.js для побудови швидкого обо масштабованого RESTful API.
  • База даних: PostgreSQL для структурованого та захищеного зберігання інформації бібліотеки.
  • Додаткові інструменти: JWT для безпечної аутентифікації користувачів та Git для контролю версій.
